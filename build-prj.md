# CLAUDE.md — Multimodal Model Inference Automation

## Project Goal

Complete the following pipeline automatically without any manual intervention:
1. Create a Python virtual environment and install dependencies
2. Download test data (images + text prompts)
3. Download an open-source multimodal model from Hugging Face or ModelScope
4. Run a full model inference pass and print results

---

## System Requirements

- OS: Linux server
- GPU: NVIDIA with CUDA support 
- Python: 
- Disk: at least 20 GB free (model weights are large)

---

## Automation Rules

**Claude Code must follow these rules throughout this project:**

1. **Use bash commands to complete every step** — do not wait for the user to run things manually
2. **Auto-fix errors immediately** — do not stop and ask unless the action is potentially destructive and intent is unclear
3. **Print a short log after each step** in the format: `[DONE] step name`
4. **Keep all files under the project root** using the structure below
5. **On network failure, automatically switch to the backup source** (HuggingFace ↔ ModelScope)
